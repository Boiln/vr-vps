# VPS Setup

1. After logging in, download the script directly to your VPS using wget.
- `wget https://raw.githubusercontent.com/Boiln/vR-VPS/main/setup_vps.sh`

2. Make the script executable:
- `chmod +x setup_vps.sh`

3. Run the script:
- `sudo ./setup_vps.sh`
